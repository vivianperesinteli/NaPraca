## 1. Value Proposition Canvas (Canvas da Proposta de Valor)

Este canvas ajuda a garantir que a solução está alinhada com as dores e ganhos do cliente. Criamos um para cada segmento de cliente.

### 1.1 Segmento: Empreendedor (MEI)

| Ganhos (Gains) | Proposta de Valor (Value Proposition) | Dores (Pains) |
| :--- | :--- | :--- |
| ✅ Ter mais clientes | **NAPRÇA - Lado Empreendedor** | ❌ Invisibilidade no mercado |
| ✅ Aumentar o faturamento | 1. **Vitrine Digital Instantânea:** Apareça em um mapa para clientes locais. | ❌ Dificuldade em atrair clientes |
| ✅ Aprender a gerir o negócio | 2. **Educação Gamificada:** Aprenda gestão com missões práticas. | ❌ Falta de conhecimento em gestão |
| ✅ Economizar tempo | 3. **Automação de Marketing:** Postagens automáticas em redes sociais. | ❌ Sobrecarga de trabalho |
| ✅ Sentir-se mais profissional | 4. **Analytics Simplificado:** Entenda o comportamento dos seus clientes. | ❌ Insegurança nas decisões |
| ✅ Conectar-se com outros | 5. **Comunidade de Empreendedores:** Troque experiências e aprenda junto. | ❌ Sentimento de solidão |

**Fit Problema-Solução:** A proposta de valor da NAPRÇA para o empreendedor ataca diretamente as dores de **invisibilidade** e **falta de conhecimento**, enquanto potencializa os ganhos de **crescimento** e **eficiência**.

### 1.2 Segmento: Consumidor Local

| Ganhos (Gains) | Proposta de Valor (Value Proposition) | Dores (Pains) |
| :--- | :--- | :--- |
| ✅ Descobrir lugares novos | **NAPRÇA - Lado Consumidor** | ❌ Dificuldade em encontrar serviços locais |
| ✅ Economizar dinheiro | 1. **Mapa Inteligente do Bairro:** Encontre tudo o que você precisa perto de casa. | ❌ Desconhecimento da oferta local |
| ✅ Apoiar a comunidade | 2. **Ofertas e Recompensas:** Ganhe descontos e pontos ao comprar localmente. | ❌ Falta de incentivos para comprar local |
| ✅ Sentir-se parte de algo | 3. **Gamificação do Consumo:** Cumpra missões e veja seu impacto no bairro. | ❌ Consumo sem propósito |
| ✅ Ter uma experiência divertida | 4. **Comunidade de Impacto:** Conecte-se com vizinhos que também apoiam o local. | ❌ Pouca conexão com a vizinhança |

**Fit Problema-Solução:** A proposta de valor para o consumidor transforma a dor da **descoberta** em um ganho de **conveniência e diversão**, alinhando o desejo de **apoio à comunidade** com incentivos tangíveis.

## 3. MVP Canvas

Este canvas foca especificamente no Mínimo Produto Viável, alinhando a proposta, as métricas e as jornadas para o primeiro lançamento.

| Proposta de Valor do MVP | Segmentos de Clientes (Early Adopters) | Jornadas do MVP |
| :--- | :--- | :--- |
| **Para o MEI:** Tenha uma vitrine digital e aprenda sua primeira lição de negócio de forma fácil e rápida. | **Empreendedores (MEIs):** | 1. Baixa o app e faz o cadastro. |
| **Para o Consumidor:** Descubra e apoie pelo menos um negócio novo no seu bairro. | - Altamente motivados a crescer. | 2. Preenche o perfil básico do seu negócio. |
| | - Já tentaram usar tecnologia antes, sem sucesso. | 3. Completa a primeira missão ("Tire uma boa foto da sua fachada"). |
| | **Consumidores Locais:** | 4. Vê seu negócio aparecer no mapa. |
| | - Engajados com a comunidade. | 5. **Consumidor:** Baixa o app. |
| | - Gostam de experimentar novidades. | 6. Abre o mapa e vê os negócios locais. |
| | | 7. Clica em um negócio para ver o perfil. |

| Features do MVP | Métricas de Sucesso | O que vem depois? |
| :--- | :--- | :--- |
| 1. **Cadastro Simples:** (Email/senha) para ambos os perfis. | **Validação do Problema:** | - Mais missões educativas. |
| 2. **Perfil Básico do Negócio:** (Nome, endereço, foto, descrição). | - >70% dos MEIs entrevistados confirmam a dor da invisibilidade. | - Sistema de recompensas para consumidores. |
| 3. **Mapa Interativo:** Exibe os negócios cadastrados. | **Validação da Solução (MVP):** | - Chat entre cliente e negócio. |
| 4. **Uma Missão Gamificada:** Para o empreendedor. | - **Retenção D7:** >40% (usuários voltam após 7 dias). | - Integração com redes sociais. |
| 5. **Analytics Básico:** (Contagem de visualizações no perfil). | - **Ativação:** >50% dos MEIs completam a primeira missão. | - Planos de assinatura (monetização). |
| | - **Engajamento:** >60% dos consumidores exploram o mapa. | - Dashboard de analytics para MEIs. |
| | - **NPS:** > 7/10. | |

| Custos e Cronograma | Riscos e Mitigações |
| :--- | :--- |
| **Custos:** | **Risco 1: Ninguém se cadastra (problema do ovo e da galinha).** |
| - **Desenvolvimento:** 2-3 desenvolvedores (8-12 semanas). | - **Mitigação:** Focar em um único bairro inicialmente; fazer aquisição manual de 20-30 MEIs antes de lançar para consumidores. |
| - **Infraestrutura:** Firebase (gratuito no início), Google Maps API (custo baixo). | **Risco 2: Empreendedores não veem valor na educação.** |
| **Cronograma:** | - **Mitigação:** A primeira missão deve gerar um resultado tangível e imediato (ex: uma foto melhor, uma descrição mais atrativa). |
| - **Semanas 1-4:** Design (UX/UI) e Setup do Projeto. | **Risco 3: Consumidores não têm motivo para voltar.** |
| - **Semanas 5-10:** Desenvolvimento (Sprints). | - **Mitigação:** Lançar com ofertas exclusivas dos primeiros MEIs cadastrados; criar uma campanha de "descubra seu bairro". |
| - **Semanas 11-12:** Testes e Lançamento Alfa. | **Risco 4: Baixa qualidade dos dados inseridos pelos MEIs.** |
| | - **Mitigação:** Criar um processo de cadastro guiado e simples; as próprias missões servem para melhorar a qualidade do perfil. |
